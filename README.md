# PRAT

Prat The Best :D

## Installation

- Be sure you have installed Python, Django

- Requirements:

    `pip install pytz`

    `pip install django-timezone-field`

    `pip install django_compressor`

- Clone the project

    ` git clone https://github.com/theadriann/prat.git `




## Usage

- Get the database ready

    ` ./manage.py makemigrations `

    ` ./manage.py migrate `

- Start Server

    ` ./manage.py runserver `
